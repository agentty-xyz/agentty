# UI Pages

Full-screen page implementations implementing the `Page` trait.

## Directory Index
- [diff.rs](diff.rs) - Diff view page.
- [health.rs](health.rs) - Health check splash screen page.
- [mod.rs](mod.rs) - Page module exports.
- [roadmap.rs](roadmap.rs) - Roadmap view page.
- [session_chat.rs](session_chat.rs) - Main chat interface page (handles both new sessions and replies).
- [session_list.rs](session_list.rs) - List of active sessions page.
